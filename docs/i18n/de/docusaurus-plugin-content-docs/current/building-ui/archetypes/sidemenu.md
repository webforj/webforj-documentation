# SideMenu-Archetyp
<!-- vale on -->

Für Projekte, die ein strukturiertes Navigationssystem benötigen, ist das `sidemenu`-Archetyp ein großartiger Ausgangspunkt. Dieses Archetyp enthält ein Seitenmenü und einen Inhaltsbereich und wurde entwickelt, um Ihnen zu helfen, Apps mit einer klaren und intuitiven Navigationsstruktur zu erstellen, die es den Benutzern erleichtert, verschiedene Teile Ihrer App zu finden und darauf zuzugreifen.

:::tip Verwendung von startforJ
Für mehr Kontrolle über Anpassung und Konfiguration können Sie [startforJ](https://docs.webforj.com/startforj/) verwenden, um Ihr Projekt zu erstellen – wählen Sie einfach das `SideMenu`-Archetyp bei der Auswahl der Konfigurationsoptionen aus.
:::
