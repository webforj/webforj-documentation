Standardmäßig können webforJ-Projekte ausgeführt werden, jedoch wird kein Lizenzzertifikat damit verbunden sein. Wenn ein nicht lizenziertes Projekt ausgeführt wird, wird ein Wasserzeichen in der Benutzeroberfläche der laufenden App angezeigt, bis ein gültiges Lizenzzertifikat eingegeben wurde.

Für einzelne Entwickler ist es einfach, ein Testlizenzzertifikat von webforJ zu erhalten. Besuchen Sie die [webforJ Preisseite](https://webforj.com/pricing/), wählen Sie den entsprechenden Plan aus und folgen Sie den Anweisungen, um eine Lizenz zu beantragen.

Um einer webforJ-App zu ermöglichen, eine Lizenz von einem Dienst, der auf Ihrem Entwicklungsrechner oder internen Netzwerk läuft, anzufordern, gehen Sie zu [Lokale BLS-Lizenz](/docs/configuration/local-bls-license).
