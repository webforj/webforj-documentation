Par défaut, les projets webforJ peuvent être exécutés mais ne disposeront pas d'un certificat de licence associé. Lors de l'exécution d'un projet non licencié, un filigrane s'affichera sur l'interface utilisateur de l'application en cours d'exécution jusqu'à ce qu'une licence valide ait été saisie.

Pour les développeurs individuels, obtenir un certificat de licence d'essai pour webforJ est simple. Visitez la [page de tarification de webforJ](https://webforj.com/pricing/), choisissez le plan approprié et suivez les étapes pour demander une licence.

Pour permettre à une application webforJ de demander une licence à un service s'exécutant sur votre machine de développement ou votre réseau interne, rendez-vous sur [Local BLS License](/docs/configuration/local-bls-license).
