Deze stap-voor-stap tutorial begeleidt je door het proces van het bouwen van een customer management app met webforJ en Spring Boot. Het leert je hoe je een moderne, gebruiksvriendelijke interface kunt maken voor het bekijken, toevoegen en bewerken van klantgegevens.

Elke stap introduceert nieuwe concepten en resulteert in een uitvoerbare Spring Boot app (JAR). Je kunt je app lokaal starten met Maven en ermee interageren in een webbrowser. Met deze opzet krijg je een snelle ontwikkelcyclus met frontend bundeling en een productieklare implementatiemodel, gebruikmakend van de embedded server van Spring Boot.

Geen voorafgaande ervaring met Spring Boot of webforJ is nodig, maar je zou een basisbegrip van Java en Maven moeten hebben om het meeste uit deze tutorial te halen. Deze tutorial behandelt Spring-concepten wanneer ze opkomen, maar degenen die geïnteresseerd zijn in een diepgaand begrip van Spring kunnen naar de [hoofddocumentatie van Spring](https://spring.io/learn) en de documentatie van Spring over [Spring Boot](https://docs.spring.io/spring-boot/index.html) gaan.

## Tutorialconcepten {#tutorial-concepts}

Het eerste deel van de tutorial is gewijd aan [projectsetup](/docs/introduction/tutorial/project-setup) om je Spring Boot + webforJ-omgeving voor te bereiden. Door de stappen te volgen, krijg je een duidelijk begrip van hoe een app evolueert terwijl je functies implementeert.

## Vereisten {#prerequisites}

Je zou de volgende tools/hulpmiddelen op je ontwikkelingsmachine moeten hebben:

- Java 21 of 25
- Maven
- Een Java IDE
- Git (aanbevolen maar niet verplicht)
