Deze stapsgewijze tutorial leidt je door het proces van het bouwen van een klantbeheertoepassing met webforJ en Spring Boot. Het leert je hoe je een moderne, gebruiksvriendelijke interface kunt maken voor het bekijken, toevoegen en bewerken van klantgegevens.

Elke stap introduceert nieuwe concepten en resulteert in een uitvoerbare Spring Boot-app (JAR). Je kunt je app lokaal opstarten met Maven en ermee interageren in een webbrowser. Met deze opzet krijg je een snelle ontwikkelingscyclus en een productieklaar implementatiemodel, met gebruik van de ingebedde server van Spring Boot.

Geen eerdere ervaring met Spring Boot of webforJ is nodig, maar je zou een basisbegrip van Java en Maven moeten hebben om het meeste uit deze tutorial te halen. Deze tutorial behandelt Spring-concepten naarmate ze opkomen, maar diegenen die geïnteresseerd zijn in een diepgaand begrip van Spring kunnen naar de [hoofddocumentatie van Spring](https://spring.io/learn) en de documentatie van Spring over [Spring Boot](https://docs.spring.io/spring-boot/index.html) gaan.

## Tutorialconcepten {#tutorial-concepts}

Door mee te volgen, krijg je een duidelijk begrip van hoe een app zich ontwikkelt naarmate je functies implementeert.

## Vereisten {#prerequisites}

Je zou de volgende tools/hulpmiddelen op je ontwikkelingsmachine moeten hebben:

- Java 21 of 25
- Maven
- Een Java IDE
- Git (aanbevolen maar niet vereist)

Controleer het [vereistenartikel](/docs/introduction/prerequisites) voor een meer gedetailleerd overzicht van de vereiste tools voor je ontwikkelomgeving.
