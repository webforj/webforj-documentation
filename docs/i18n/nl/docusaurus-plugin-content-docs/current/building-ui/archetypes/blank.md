# Leeg archetype

De `blank` archetype is een fundamentaal startproject voor webforJ-toepassingen. Deze sjabloon biedt een schone lei om je app helemaal vanaf nul op te bouwen. Het is ideaal voor ontwikkelaars die volledige controle willen over de structuur en componenten van hun app zonder enige vooraf gedefinieerde beperkingen.

:::tip Gebruik startforJ
