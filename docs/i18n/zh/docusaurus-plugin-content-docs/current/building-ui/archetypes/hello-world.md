# HelloWorld 原型
<!-- vale on -->

该原型创建一个简单的 hello world 应用程序，以演示使用 webforJ 构建 UI 的基础知识。此模板非常适合初学者快速入门。它提供了一个简单明了的示例，展示如何设置和运行基本的 webforJ 应用程序，使其成为新开发人员的绝佳起点。

:::tip 从头开始
该原型创建一个具有少量组件和一些样式的简约应用程序。对于希望创建最少脚手架的项目的开发人员，请参见 [`blank` 原型](./blank)。
:::

:::tip 使用 startforJ
为了获得更好的自定义和配置控制，您可以使用 [startforJ](https://docs.webforj.com/startforj/) 创建您的项目 - 只需在选择配置选项时选择 `HelloWorld` 原型即可。
:::
