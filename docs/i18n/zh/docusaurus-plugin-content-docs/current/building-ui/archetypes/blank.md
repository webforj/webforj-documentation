# 空白原型

`blank` 原型是 webforJ 应用程序的基础启动项目。该模板为您从头开始构建应用程序提供了一个干净的起点。它非常适合希望对应用程序的结构和组件拥有完全控制权的开发人员，没有任何预定义的限制。

:::tip 使用 startforJ
为了更好地控制自定义和配置，您可以使用 [startforJ](https://docs.webforj.com/startforj/) 创建项目 - 在选择配置选项时选择 `Blank` 原型。
:::
