# Tabs 原型

`tabs` 启动项目生成一个具有简单标签界面的应用。适用于需要通过标签访问多个视图或部分的项目，这个原型提供了一种干净、有序的方式来管理应用的不同部分，使用户能够在各种部分之间轻松导航，而不会使用户界面显得杂乱。

:::tip 使用 startforJ
为了获得更多的自定义和配置控制，您可以使用 [startforJ](https://docs.webforj.com/startforj/) 来创建您的项目 - 只需在选择配置选项时选择 `Tabs` 原型。
:::
