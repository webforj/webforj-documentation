Por defecto, los proyectos de webforJ se pueden ejecutar, pero no tendrán un certificado de licencia asociado. Al ejecutar un proyecto sin licencia, se mostrará una marca de agua en la interfaz de usuario de la aplicación en ejecución hasta que se haya ingresado una licencia válida.

Para desarrolladores individuales, obtener un certificado de licencia de prueba de webforJ es simple. Visite la [página de precios de webforJ](https://webforj.com/pricing/), elija el plan adecuado y siga las instrucciones para solicitar una licencia.

Para permitir que una aplicación de webforJ solicite una licencia a un servicio que se ejecuta en su máquina de desarrollo o red interna, consulte [Licencia BLS Local](/docs/configuration/local-bls-license).
