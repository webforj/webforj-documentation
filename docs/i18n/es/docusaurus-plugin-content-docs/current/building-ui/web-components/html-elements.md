Mientras que la clase `Element` de webforJ permite a los usuarios crear elementos HTML en sus aplicaciones, se ha incluido un conjunto de elementos HTML estándar junto con los componentes centrales de webforJ para facilitar su uso.

Los siguientes componentes se pueden utilizar y se corresponden con los elementos HTML:

|Clase webforJ|Elemento HTML|¿Puede agregar componentes?|
|:--:|:--:|:--:|
|`Anchor`|`<a>`| ✔️ |
|`Article`|`<article>`| ✔️ |
|`Aside`|`<aside>`| ✔️ |
|`Break`|`<hr>`| ❌ |
|`Div`|`<div>`| ✔️ |
|`Emphasis`|`<em>`| ✔️ |
|`Fieldset`|`<fieldset>`| ✔️ |
|`Footer`|`<footer>`| ✔️ |
|`FormattedText`|`<pre>`| ✔️ |
|`H1 - H6`|`<h1> - <h6>`| ✔️ |
|`Header`|`<header>`| ✔️ |
|`Iframe`|`<iframe>`| ❌ |
|`Img`|`<img>`| ❌ |
|`Legend`|`<legend>`| ❌ |
|`ListEntry`|`<li>`| ✔️ |
|`Main`|`<main>`| ✔️ |
|`NativeButton`|`<button>`| ✔️ |
|`Nav`|`<nav>`| ✔️ |
|`OrderedList`|`<ol>`| ✔️ |
|`Paragraph`|`<p>`| ✔️ |
|`Section`|`<section>`| ✔️ |
|`Span`|`<span>`| ✔️ |
|`Strong`|`<strong>`| ✔️ |
|`UnorderedList`|`<ul>`| ✔️ |

Estos componentes se han implementado para proporcionar una API para interactuar con las diversas propiedades, atributos y funcionalidades que se esperan de estos elementos.

También se proporcionan métodos para agregar, eliminar y acceder a los componentes, como se detalla en esta sección de la clase [`Element`](../elements.md#component-interaction).
