# Tabs-arkkitehtuuri

`tabs`-aloitusprojekti luo sovelluksen, jossa on yksinkertainen välilehtirajapinta. Ihanteellinen projekteille, jotka vaativat useita näkymiä tai osioita, joihin pääsee käsiksi välilehtien kautta, tämä arkkitehtuuri tarjoaa siistin ja järjestäytyneen tavan hallita sovelluksen eri osia, mikä tekee erilaisten osioiden välillä navigoinnista helppoa ilman, että käyttäjärajapinta täyttyy.

:::tip Käyttäen startforJ
Saat lisää hallintaa muokkaamiseen ja konfigurointiin käyttämällä [startforJ](https://docs.webforj.com/startforj/) -työkalua projektisi luomiseen - valitse vain `Tabs`-arkkitehtuuri konfigurointivaihtoehtoja valitessasi.
:::
