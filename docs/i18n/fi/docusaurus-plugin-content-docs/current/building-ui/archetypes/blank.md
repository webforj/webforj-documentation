# Tyhjät arkkityypit

`blank` -arkkitehti on perustavanlaatuinen aloitusprojekti webforJ-sovelluksille. Tämä malli tarjoaa puhtaan alustan, jolta voit rakentaa sovellustasi alusta alkaen. Se on ihanteellinen kehittäjille, jotka haluavat täydellistä kontrollia sovelluksensa rakenteesta ja komponenteista ilman ennalta määritettyjä rajoituksia.

:::tip Käyttäen startforJ
Jos haluat enemmän hallintaa muokkauksen ja konfiguraation suhteen, voit käyttää [startforJ](https://docs.webforj.com/startforj/) -työkalua projektisi luomiseen - valitse vain `Blank` -arkkitehti, kun valitset konfiguraatioasetuksia.
:::
