Sovita sovellustasi Javalla palvelimella (katso [Asiakas ja palvelin](/docs/architecture/client-server)) sisältää kokonaisia luokkia verkkohyökkäyksistä ennen kuin kirjoitat rivin koodia. Se ei kuitenkaan poista vastuuksiasi. Nämä sivut selittävät, missä raja kulkee, mitä webforJ hoitaa puolestasi, missä pysyt vastuussa ja kuinka pitää tuotantokäyttö valmis.

Jotta voit hallita, kuka voi päästä kuhunkin näkymään, katso loppuosaa [Turvallisuus](/docs/security/overview) osiosta, joka käsittelee todennusta ja reittikohtaista valtuutusta.
